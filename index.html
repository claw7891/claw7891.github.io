AUVSI XPonential Trade Show Tools

Mobile-friendly web interfaces for navigating and documenting your experience at the AUVSI XPonential trade show.

Interface 1: Exhibitor Selector

Create a list of exhibitors with descriptions and checkboxes. Submit to get JSON of selected exhibitors.

Open Exhibitor Selector

Interface 2: Path Optimizer

Take JSON from Interface 1 to compute an efficient path through the exhibition map with visual route and ordered table.

Open Path Optimizer

Interface 3: Note Taker

Take JSON from Interface 1 to create a note-taking interface with sections per exhibitor, including notes and business card image capture.

Open Note Taker
